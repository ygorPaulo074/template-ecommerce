## Template de E-Commerce Website

Este repositório contém um template básico para um website de e-commerce. O template inclui páginas essenciais como página inicial, página de produtos, página de detalhes do produto, carrinho de compras e checkout.

### Estrutura do Projeto

O projeto será desenvolvido em estrutura modular, facilitando a manutenção e a escalabilidade.
